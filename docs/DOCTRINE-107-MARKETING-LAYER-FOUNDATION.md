# Doctrine S107 - Marketing Layer Foundation
# Source: Antonio Cook | Codified: 2026-06-28

## The strategic decision

The sales + lifecycle + customer success agent swarm is the FOUNDATION.
The marketing layer is the next tier built ON TOP of that foundation.
Marketing is what Antonio scales NEXT, once back-of-house is autonomous.

## Why marketing agents need a unified swarm beneath them

  - A content agent generating an IG Reels clip needs to know who is in BTF this month (case studies), funnel stage prospects (CTAs), pipeline gaps (offers to push)
  - An ad agent optimizing campaigns needs to know lead source attribution, LTV by entry door, conversion chain across the swarm
  - A clipping agent processing Workshop Wednesday recording needs to know which clips become Skool posts, BTF lead magnets, LaunchPad onboarding videos
  - A distribution agent posting across YT+TT+IG+LI needs platform brand voice + audience segment variants + performance feedback loop

If marketing agents had their own view of customer + offers + lifecycle, they would duplicate the swarm. Wasted effort + inconsistent state + brand confusion.

Because the swarm is unified, marketing agents READ the same brain state and WRITE feedback back into it. Every clip engagement, every ad conversion, every social comment becomes signal in the same intake pipeline.

## Tier -2: Marketing Layer (planned)

| Agent | Role | Reads from | Writes to |
|---|---|---|---|
| content_engine | Generates posts, captions, scripts | crm_orchestrator (audiences), customer_memory (winning angles) | content_orchestrator (publish queue) |
| clip_engine | Long-form > short clips | Workshop recordings, MMA classes | content_engine (clip variants by platform) |
| ad_engine | Manages paid campaigns Meta+Google+TikTok | revenue_orchestrator (target offers), pipeline_health (gaps) | ad platforms (creative + budget) |
| distribution_engine | Posts across platforms | content_engine assets, brand-voice playbooks | YT/IG/TT/LI APIs |
| attribution_engine | Traces leads back to source content/ad | sales_dept closes, all intake logs | revenue_orchestrator (LTV by source) |
